部署静态文件

当 Django 使用在正式环境的时候，静态文件需要被托管于 Web 服务器。

在 Kont BS 中使用 Makefile 可以快速导出静态文件到项目目录下的 staticfiles 的文件夹中

make collection

uWsgi 部署

Django 默认支持 uwsgi 可以方便的启动，在 Makefile 添加一些能增强性能的配置，使用 Makefile 也可以迅速启动项目。

make start-uwsgi

如果想停止可以使用：

make stop-uwsgi

如果想重启 uwsgi 可以使用：

make reload-uwsgi
